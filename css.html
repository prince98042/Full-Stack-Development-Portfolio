CSS Styles Only & Explanation

body {
    background-color: #f0f0f0;
    color: #333;
    font-family: sans-serif;
}

h1 {
    color: teal;
}

body — Applies global styles to the entire page:

  • background-color: #f0f0f0; – Sets the background to a light gray tone.
  • color: #333; – Changes text color to dark gray for good contrast.
  • font-family: sans-serif; – Applies a clean, modern sans-serif font.

h1 — Styles the heading:

  • color: teal; – Gives the heading a teal color for emphasis and visual appeal.

This CSS is written using the internal style method, where styles are placed inside a <style> tag within the document's <head> section.
