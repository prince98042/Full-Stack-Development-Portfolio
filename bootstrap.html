Bootstrap Code & Explanation

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Bootstrap Example</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body class="p-5">
    <div class="container text-center">
        <h1 class="text-primary">Bootstrap Example</h1>
        <p>This is styled using Bootstrap's utility classes and components.</p>
        <button class="btn btn-success">Click Me</button>
    </div>
</body>
</html>

<link href=...bootstrap.min.css> — Imports the Bootstrap 5 CSS framework.

class="p-5" — Adds padding (spacing) to the body using Bootstrap’s utility class.

class="container" — Centers and contains content with fixed width and padding.

class="text-center" — Aligns text to the center.

class="text-primary" — Applies Bootstrap’s primary theme color (blue) to the heading.

class="btn btn-success" — Styles the button with Bootstrap’s green “success” theme.

This example uses **Bootstrap 5 utility classes** for layout, colors, spacing, and buttons without writing custom CSS.
